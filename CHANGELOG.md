# 📝 Changelog
Todos los cambios notables en este proyecto serán documentados en este archivo.

## [1.5.0] - 2026-05-11 (Titanium Release)
### Añadido
- **Certificación Maestra**: Auditoría final de 112,000 transacciones completada.
- **Dashboard Premium**: Interfaz visual de terminal con colores y barras de progreso.
- **Modo Extreme**: Soporte para uso total de núcleos de CPU.

## [1.2.0] - 2026-05-11 (Network Phase)
### Añadido
- **Motor Remoto**: Implementación de Sonda de Producción (HTTP/gRPC).
- **Algoritmo Ramp-up**: Búsqueda automática de límites de infraestructura.
- **Métricas P99**: Cálculo de latencia avanzada.

## [1.1.0] - 2026-05-10 (Concurrency Phase)
### Añadido
- **Orquestador Multihilo**: Uso de `Worker Threads` para estrés masivo.
- **Bóveda Atómica**: Sincronización mediante `SharedArrayBuffer` y `Atomics`.
### Corregido
- Vulnerabilidad de *Double-Spend* detectada en auditoría de Fase 2.

## [1.0.0] - 2026-05-10 (Crypto Phase)
### Añadido
- **Motor Criptográfico**: Implementación de AES-256-GCM para integridad de ADN.
- **Lógica de Quema**: Primera implementación de tokens inmutables.
